# SQL Portfolio — аналитические запросы

Набор SQL-запросов, демонстрирующих навыки работы с данными.

## Содержание

| Файл | Что делает | Ключевые конструкции |
|------|-----------|---------------------|
| `grouping_aggregates.sql` | Анализ успешности доставки по дням недели | `FILTER`, `DATE_PART`, `TO_CHAR` |
| `joins.sql` | Кто заказывал и доставлял самые большие заказы | `CTE`, `AGE`, `LEFT JOIN`, подзапросы |
| `subqueries.sql` | Поиск заказов с топ-5 дорогими товарами | `UNNEST` (работа с массивами), `IN` |
| `window_functions.sql` | Накопительная сумма заказов по дням | `SUM() OVER (ORDER BY date)` |

## Используемые технологии

- PostgreSQL-совместимый SQL
- Оконные функции
- CTE (Common Table Expressions)
- Работа с массивами (`UNNEST`)
- Агрегация с `FILTER`

## Контекст

Запросы написаны для учебного проекта. Реальные дашборды в Redash строились на основе подобной логики.
