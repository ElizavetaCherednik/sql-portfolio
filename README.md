# SQL Portfolio — аналитические запросы

**Цель проекта:** Показать навыки SQL, необходимые аналитику данных для решения реальных бизнес-задач (логистика, продажи, ассортимент).

**Что умею:** оконные функции, CTE, работу с массивами (UNNEST), агрегацию с FILTER, JOIN всех типов, анализ по датам.

## Содержание

| Файл | Что делает | Ключевые конструкции |
|------|------------|----------------------|
| grouping_aggregates.sql | Анализ успешности доставки по дням недели → какие дни самые проблемные для логистики | FILTER, DATE_PART, TO_CHAR |
| joins.sql | Кто заказывал и доставлял самые большие заказы → сегментация клиентов и курьеров | CTE, AGE, LEFT JOIN, подзапросы |
| subqueries.sql | Поиск заказов с топ-5 дорогими товарами → анализ ассортимента | UNNEST, IN |
| window_functions.sql | Накопительная сумма заказов по дням → динамика продаж для дашборда | SUM() OVER (ORDER BY date) |

## Используемые технологии
- PostgreSQL-совместимый SQL
- Оконные функции
- CTE (Common Table Expressions)
- Работа с массивами (UNNEST)
- Агрегация с FILTER
